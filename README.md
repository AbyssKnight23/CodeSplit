# CodeSplite

CodeSplite es un desensamblador educativo desarrollado en Python.

## Características

- Detección de ejecutables PE (Windows)
- Detección de ejecutables ELF (Linux)
- Soporte para arquitecturas x86 (32 bits)
- Soporte para arquitecturas x64 (64 bits)
- Visualización hexadecimal
- Extracción de la sección .text
- Desensamblado de instrucciones comunes

## Tecnologías

- Python 3
- Tkinter
